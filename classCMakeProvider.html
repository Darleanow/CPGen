Detailed Description

Static factory that generates individual CMake script snippets.

Each method corresponds to one logical CMake construct. The caller is responsible for ordering the snippets correctly in the output file.

Typical assembly order

  1. getMinimumRequired
  2. getProjectHeader
  3. getStandard
  4. getIncludeFetchContent *(if any FetchContent modules)*
  5. getDeclareModule *(once per module)*
  6. getMakeAvailable *(once, with all module names)*
  7. getCompileOptions
  8. getClangTidyTooling *(if clang-tidy is enabled)*
  9. getTargetDeclaration *(once per target)*
  10. getIncludeDirectories *(once per target)*
  11. getTargetLink *(once per target that has dependencies)*
  12. getEnableTests *(if a test target is present)*
