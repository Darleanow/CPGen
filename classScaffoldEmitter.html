Detailed Description

Generates starter source files for a new C++ project.

Produces:

  • src/main.cpp — hello-world entry point.
  • tests/test_main.cpp — basic test skeleton (when a test target exists).
  • .gitignore — standard C++ ignores (when Git is enabled).
  • README.md — minimal project readme.
